<nav>- is used to define the navigation bar of the webpage
<favicon>- it is the shortcut icon of the webpage
<form>- is used to input data on the webpage 
<div>- is used for defining a section of your document. With the <div> tag, you can group large sections of HTML elements together and format them with CSS
<textarea>- is used to input long user data 
<input>- is used to determine what kind of input is to be used in the form
<iframe>- is used to embed or add youtube videos in the website 
<footer>- is used to define the footer section of the webpage
